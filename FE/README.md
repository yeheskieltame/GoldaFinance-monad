# Golda Finance — Frontend

This is the Next.js frontend for Golda Finance. For full project documentation, architecture details, and deployment information, refer to the root README.

Live: https://golda-finance-monad.vercel.app/

## Setup

```bash
pnpm install
cp .env.example .env.local  # fill in your keys
pnpm dev
```

Required environment variables:

```env
NEXT_PUBLIC_PRIVY_APP_ID=
NEXT_PUBLIC_GEMINI_API_KEY=
NEXT_PUBLIC_X402_PAYEE=
X402_SERVICE_PRIVATE_KEY=
```
